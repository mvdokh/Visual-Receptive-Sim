rgbtolms.py  -- Python script to convert RGB files entered on the
	        command line (type has to be readable by PIL) to three
	        PGM files named xxx_lms_0.pgm, xxx_lms_1.pgm,
	        xxx_lms_2.pgm which are the L/M/S cone activities
	        using the source functions in RGB.txt and LMS.txt.

rgbtolms.pdf -- Technical report summarizing what rgbtolms.py does.

rgbtolms.h   -- Auto-generated by rgbtolms.py.  C Header file with the
		processed data sets to allow easy conversion within C
		code.

rgbtolms.c   -- Function to convert an RGB pixel to LMS pixel in C.
		Requires rgbtolms.h

/sources/*   -- Source files used for LMS.txt and RGB.txt files.

Other scrips that are useful but not written by me:

pilconvert.py -- A script provided with PIL that will convert files
	         between types using the PIL Python library.  
rgb3toppm     -- Converts R/G/B PGM files back to a PPM file.  Use
	         this to go back to the original images if needed.
		 Assumes that the source B/W are RGB and not the LMS.
